Activating Strict Mode in JS

What is Strict Mode?

Ans: It use to write a secure code in java script.
basically it's avoid us to introduce bugs into our code.
eg:- developers write more reliable and error-free code by catching common coding mistakes and "unsafe" actions. like below:

  • Variables must be declared
  • Assignment to undeclared variables is not allowed: (In strict mode, assigning a value to a variable
    that has not been declared will result in a ReferenceError.)
  • The this keyword behaves differently:
    In non-strict mode, when a function is called without an explicit this context, it defaults to the global object (e.g., window in a web browser). In strict mode, this is set to undefined in such cases, which can help catch potential bugs.

  • And much more like this

How to activate strict Mode in JS code ?

To enable strict moder in javaScript add the following line at the top of your script or function.
"use strict";
